# API Reference Guide

This is the Python public API for controlling Metronomo, the clock signal
generator and distributor by Intermodulation Products.

The {class}`.Metronomo` class is the usual entry point and is available from the
top module:
```python
from metronomo import Metronomo
```

Detailed information is provided in the documentation for each module:
